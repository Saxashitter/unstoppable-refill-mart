# UNSTOPPABLE: Refill Mart

Open source, free, and challenging platformer based on retro games, Touhou, Sonic Robo Blast 2, Pizza Tower and more.
This game is currently **in a conceptual state**, but we are working on bringing this game to life very slowly...

## How to Contribute
This game is made in **Godot Engine 4**. If you know Godot, and want to implement a new feature, contact me on Discord **(@saxashitter)** to either be a coder for this game, or help with implementation of said feature.
Alternatively, if you wanna sprite, feel free to do so.

Pull requests will only be accepted if it's a bugfix or optimization.

## How to Test
Simply clone the repo, open Godot, import the project and run as if it wwas a normal project. Tinker with it if you want.
